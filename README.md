## ✨ Features
- **Timers in the system tray** (Windows 10/11)  
- **Multiple independent timers** support  
- **Hotkeys with modifiers** (`Ctrl`, `Alt`, `Shift`)  
- **Sound notifications** (WAV format)  
- **Alerts on start, stop, and finish**  

---

## 🚀 Quick Start
- **Download and run the app** — it will appear in the system tray.  
- By default, **one 5‑minute timer** is created (`Shift+T`).  
- **Add a new timer** via the tray menu (**middle mouse button → ➕ Add**).  

---

## 🌐 Localization
Available languages:  
- **UA** — Ukrainian  
- **EN** — English  
- **RU** — Russian  

---

## ⚙️ Configuration
Settings are stored in:  
`%LOCALAPPDATA%\TimerApp\settings.json`  

⚠️ **Important**: Each timer must have a **unique hotkey combination**!
